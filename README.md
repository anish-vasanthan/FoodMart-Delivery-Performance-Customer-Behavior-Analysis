# FoodMart Delivery Performance & Customer Behavior Analysis

A full-stack data analytics capstone project analyzing 100,000+ food 
delivery orders across 7 Indian cities using Excel, MySQL, and Power BI.

---

## Problem Statement

Food delivery platforms generate massive volumes of data but struggle 
to identify performance issues like delivery delays, low customer 
ratings, and restaurant inefficiencies. This project builds a 
data-driven solution to analyze delivery performance, customer 
behavior, and operational challenges.

---

## Project Objectives

- Analyze delivery performance and customer behavior
- Identify peak order periods and revenue trends
- Evaluate restaurant and delivery partner efficiency
- Detect issues affecting customer satisfaction
- Provide actionable insights via interactive Power BI dashboard

---

## Dataset Overview

| Table | Records | Key Columns |
|-------|---------|-------------|
| Customers | 10,000 | customer_id, name, city, signup_date |
| Restaurants | 500 | restaurant_id, name, cuisine_type, city |
| Delivery Partners | 1,000 | partner_id, name, joining_date |
| Orders | 100,000 | order_id, order_value, delivery_time, rating |
| Complaints | ~5,000 | complaint_id, complaint_type, resolution_time |

**Cities covered:** Bangalore, Chennai, Delhi, Hyderabad, Kolkata, 
Mumbai, Pune  
**Cuisine types:** Continental, Chinese, American, Indian, Mexican, 
Thai, Italian
